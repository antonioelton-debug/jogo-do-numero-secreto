## 📝 Descrição do Projeto
Este é um projeto desenvolvido durante o curso de **Lógica de Programação** da Alura. O objetivo é criar um jogo interativo onde o usuário deve descobrir um número secreto gerado aleatoriamente pelo sistema.

O projeto foca na manipulação de elementos HTML através do JavaScript, uso de funções com parâmetros e retorno, além de estruturas de controle de fluxo e listas (arrays).

## 🚀 Funcionalidades
* Geração de um número aleatório entre 1 e 10 (ou o limite definido).
* Sistema de tentativas com contador.
* Dicas visuais e sonoras (opcional) que informam se o número secreto é maior ou menor que o chute.
* Reinicialização do jogo ao acertar, limpando o campo de entrada e gerando um novo número.
* **Lógica de Listas:** O jogo armazena números já sorteados para que não se repitam até que todos os números da lista sejam usados.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estrutura da página.
* **CSS3**: Estilização e layout (Design Líquido).
* **JavaScript**: Lógica do jogo e interatividade.
